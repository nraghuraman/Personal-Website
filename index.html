Cross-Image Context Matters for Bongard Problems

Current machine learning methods struggle to solve Bongard problems, which are a type of IQ test that requires deriving an abstract “concept” from a set of positive and negative “support” images, and then classifying whether or not a new query image depicts the key concept. In this research project with collaborators at the Stanford AI Lab (SAIL), I explored a few approaches for incorporating cross-image context within Bongard problems in order to obtain improved performance. Our top-performing approach sets a new state-of-the-art on two benchmark Bongard datasets.
